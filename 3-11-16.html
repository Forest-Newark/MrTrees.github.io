Text Formating 2 - SubScript -"sub" || SuperScript - "sup" || Code Font - "code"

Pre -      Formated         Text
     Keeps       all             Spaceing            - "pre"

Text color - "style = 'color: red;'"

Background Color - "style = 'background-color:yellow;'"

Font weight - "style = 'font-weight:bold;'

Border - "style = 'border: 3px solid red';"

Padding - "style = 'padding:5px;'
